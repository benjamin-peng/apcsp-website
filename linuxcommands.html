Linux Commands

Command Definition Example
ls Lists the files in the current directory. ls
cd Changes directory. cd directoryname
mkdir Makes a new directory. mkdir directoryname
rmdir Removes a directory. rmdir directoryname
nano Open the nano file editor. nano filename
apt-get Installs a package. sudo apt-get chromium
chmod Makes a file executable. sudo apt-get chromium
ping Checks the connection of an IP address. ping google.com
kill Ends a process. clear
cat Reads a file. cat filename
